Motion

- What is meant by motion?

An object is said to be in motion when it changes it position with respect to time or a reference point.


- Example

If I am walking my position is chaning with respect to time thus I am in motion.


- What is rest?

An object is said to be at rest if it does not cahnge it position with respect to time or a reference point.
